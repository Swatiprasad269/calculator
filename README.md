# 🧮 Calculator Web App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations with a clean user interface.

---

## 🚀 Live Demo

👉 https://swatiprasad269.github.io/calculator/

---

## 📌 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧼 Clear button
* 📱 Responsive design

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
calculator/
│
├── index.html
├── calculator.css
└── (optional) script.js
```

---

## 💡 How It Works

* User clicks buttons to input numbers and operators
* JavaScript handles the calculations
* Result is displayed instantly on the screen
